# Dutton Cavanaugh Website - Local Server

A simple Express server to host the Dutton Cavanaugh website locally.

## Setup Complete! ✅

The website is already set up and ready to run. All files from www.duttoncavanaugh.com have been saved to the `public/` directory.

## Usage

Start the server:
```bash
npm start
```

The website will be available at: **http://localhost:3000**

To stop the server: Press `Ctrl + C` in the terminal

## Alternative: Run the scraper

If you want to try the automated scraper again:
```bash
npm run scrape
```

Note: The website may block automated scraping attempts.

## Project Structure

```
duttoncavanaugh/
├── public/           # Website files (HTML, CSS, JS, images)
├── server.js         # Express server
├── scraper.js        # Automated scraper (may not work due to site protection)
└── package.json      # Node.js dependencies
```
